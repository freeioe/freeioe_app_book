# SHA-256哈希算法 (hashings.sha256)

SHA-256（Secure Hash Algorithm 256-bit）是SHA-2家族的一员，产生256位（32字节）的哈希值。它是目前最广泛使用的安全哈希算法之一。

## 特性

- **摘要长度**：256位（32字节）
- **块大小**：512位（64字节）
- **输出格式**：通常表示为64个十六进制字符
- **安全性**：目前认为安全，无已知实用碰撞攻击

## 使用方法

```lua
local sha256 = require 'hashings.sha256'

-- 创建SHA-256哈希对象并计算哈希
local h = sha256:new('Hello World')
print(h:hexdigest())  -- 输出: A591A6D40BF420404A011733CFB7B190D62C65BF0BCDA32B57B277D9AD9F146E

-- 流式处理
local h = sha256:new()
h:update('First part ')
h:update('Second part')
print(h:hexdigest())

-- 获取二进制输出
local h = sha256:new('Hello World')
local binary = h:digest()  -- 32字节二进制数据
```

## 接口说明

### new

创建SHA-256哈希对象

#### 函数原型

```lua
function sha256:new(data)
end
```

#### 参数说明

* data
  可选，初始数据字符串

#### 返回值

返回SHA-256哈希对象

### update

更新哈希计算状态

#### 函数原型

```lua
function sha256:update(data)
end
```

#### 参数说明

* data
  要添加到哈希计算中的数据字符串

### digest

获取二进制哈希摘要

#### 函数原型

```lua
function sha256:digest()
end
```

#### 返回值

返回32字节的二进制字符串

### hexdigest

获取十六进制哈希摘要

#### 函数原型

```lua
function sha256:hexdigest()
end
```

#### 返回值

返回64个字符的十六进制字符串（大写）

### copy

复制哈希对象及其状态

#### 函数原型

```lua
function sha256:copy()
end
```

#### 返回值

返回包含相同内部状态的新SHA-256对象

## 使用示例

### 文件完整性验证

```lua
local sha256 = require 'hashings.sha256'

local function file_sha256(filename)
    local file = io.open(filename, 'rb')
    if not file then return nil, 'Cannot open file' end

    local h = sha256:new()
    while true do
        local block = file:read(64 * 1024)  -- 64KB块
        if not block then break end
        h:update(block)
    end
    file:close()
    return h:hexdigest()
end

local hash = file_sha256('important.dat')
print('SHA-256:', hash)
```

### 密码哈希（带盐）

```lua
local sha256 = require 'hashings.sha256'

local function hash_password(password, salt)
    local h = sha256:new()
    h:update(salt)
    h:update(password)
    return h:hexdigest()
end

-- 使用随机盐（更好的方法是使用PBKDF2）
local salt = 'random_salt_value'
local hashed = hash_password('user_password', salt)
```

### 数据签名准备

```lua
local sha256 = require 'hashings.sha256'

local function prepare_signature(data, secret_key)
    local h = sha256:new()
    h:update(data)
    h:update(secret_key)
    return h:hexdigest()
end

local signature = prepare_signature('transaction_data', 'secret_key')
```

### 创建唯一标识符

```lua
local sha256 = require 'hashings.sha256'

local function generate_id(content)
    local h = sha256:new(content)
    local hex = h:hexdigest()
    -- 使用前16个字符作为ID
    return hex:sub(1, 16)
end

print(generate_id('user@example.com'))
```

## 性能特点

- **计算速度**：快，但比MD5/SHA-1慢
- **内存占用**：中等（64字节缓冲区 + 8个32位状态变量）
- **安全性**：高，目前无已知实用碰撞攻击

## 类属性

```lua
sha256.digest_size  -- 32 (字节)
sha256.block_size   -- 64 (字节)
```

## 安全建议

### 适用场景

✅ **推荐用于**：
- 数据完整性验证
- 数字签名
- 密码哈希（配合盐和迭代）
- 消息认证码（HMAC）
- 区块链和加密货币
- 证书和证书指纹

### 不适用场景

❌ **不推荐用于**：
- 需要极高性能的非安全哈希表（使用MD5或自定义哈希）
- 密码直接存储（应使用PBKDF2/bcrypt/scrypt）

## 与其他算法比较

| 算法 | 摘要长度 | 速度 | 安全性 |
| :--- | :--- | :--- | :--- |
| MD5 | 128位 | 最快 | 已破解 |
| SHA-1 | 160位 | 快 | 已破解 |
| **SHA-256** | **256位** | **中等** | **安全** |
| SHA-512 | 512位 | 慢 | 安全 |
| BLAKE2b | 可变 | 快 | 安全 |

## 实现细节

SHA-256工作原理：
1. **消息填充**：将消息填充到512位的倍数
2. **初始化**：使用8个32位初始哈希值
3. **消息处理**：以512位块为单位处理
4. **压缩函数**：使用64轮压缩函数
5. **输出**：组合8个32位哈希值得到256位摘要

对于大多数应用，这些实现细节是透明的，开发者只需使用提供的接口。
