# 缓冲区模块集合

本目录包含FreeIOE中使用的各种缓冲区模块，用于不同的数据处理场景。

## 模块列表

| 模块 | 说明 |
| :--- | :--- |
| [threshold](threshold.md) | 阈值缓冲区，用于事件频率控制 |
| [cycle](cycle.md) | 循环缓冲区，固定大小自动丢弃最老数据 |
| [period](period.md) | 批次数据缓冲区，周期性批量处理 |

## 使用场景

### threshold (阈值缓冲区)
适用于需要控制事件触发频率的场景，例如：
- 限制特定时间窗口内的事件数量
- 防止告警风暴
- 流量控制

### cycle (循环缓冲区)
适用于需要固定大小缓冲的场景，例如：
- 数据队列管理
- 临时数据缓存
- 日志缓冲

### period (批次缓冲区)
适用于需要批量处理的场景，例如：
- 批量数据上报
- 周期性数据同步
- 批量写入操作
